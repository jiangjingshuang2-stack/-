## 目录分类

- `final_rerun_20260606_172023/`：最终统一完整实验。
- `final_rerun_20260606_172023_results_only.zip`：最终结果压缩包。
- `final_rerun_20260606_171808/`：仅运行 3 次迭代的冒烟测试。
- `final_rerun_20260606_171846/`：中间完整运行版本。
- `synthetic/`：早期合成数据扩展实验。
- `diabetes/`、`wine_red/`、`communities_crime/`：早期真实数据单项图片。
- `.matplotlib_cache/`：程序缓存，不属于实验结果。

最终目录中的每个子文件夹均有 README，介绍当前结果、图片用途、主要结论以及它与其他实验之间的关系。
