# Lambda 敏感性实验说明

本实验固定合成数据和其他参数，令 `lambda` 在 `0.01, 0.02, 0.05, 0.1, 0.2, 0.5, 1.0` 中变化，比较四种算法的表现。

## 文件与图片

### `lambda_nnz.png`

展示不同 lambda 下各算法最终解的非零系数数量。

- 用途：分析正则化强度如何影响模型稀疏度。
- 读图：nnz 越小，模型越稀疏。
- 主要结论：对 ISTA、FISTA 和 ADMM，lambda 从 `0.01` 增至 `1.0` 时，nnz 大致从 `63` 降至 `2`。Subgradient 的 nnz 始终约为 `200`，说明其有限迭代结果难以产生严格零系数。
- 报告/PPT 建议：用于说明“lambda 越大，稀疏性越强”。

### `lambda_recovery_error.png`

展示不同 lambda 下的系数恢复误差。

- 用途：寻找稀疏性和恢复准确度之间的平衡。
- 主要结论：ISTA、FISTA 和 ADMM 在 `lambda=0.05` 时恢复误差最低，约为 `0.343`；lambda 太小会保留过多噪声特征，lambda 太大则会过度压缩真实系数。
- 报告/PPT 建议：用于解释正则化参数需要调优，不能只追求更稀疏。

### `lambda_objective.png`

展示各 lambda 下算法最终目标函数值。

- 用途：比较不同算法在同一个 lambda 下是否到达一致解。
- 注意：不同 lambda 对应不同目标函数，因此不应简单认为跨 lambda 的目标值越小越好。
- 主要结论：同一 lambda 下，ISTA、FISTA 和 ADMM 的最终目标值几乎一致，验证了三种算法实现的一致性；Subgradient 的目标值通常更高。

### `lambda_sensitivity.csv`

保存全部 `7 × 4 = 28` 组实验的目标值、恢复误差、nnz、迭代次数和运行时间。

## 可直接使用的结论

> Lambda 控制 LASSO 的稀疏性与拟合能力之间的权衡。本实验中，lambda=0.05 获得最低恢复误差，而更大的 lambda 会进一步减少非零系数，但同时造成明显的欠拟合。

## 与其他实验如何串联

- 基础实验使用 `lambda=0.1`，得到 recovery error 约 `0.390`、nnz 为 `17`。本实验说明该参数是在误差与稀疏度之间做出的折中。
- `lambda=0.05` 虽然恢复误差最低，但 nnz 为 `36`；若只追求误差，可选更小 lambda，若更重视稀疏性，则应增大 lambda。
- ISTA、FISTA、ADMM 在每个 lambda 下的结果高度接近，与 `sklearn_compare/` 一起支持算法实现正确。
- 真实数据实验中大部分特征仍被保留，说明合成数据上的 lambda 经验不能机械套用到所有真实数据，应结合验证集重新选择。
