# 多随机种子重复实验说明

本实验使用随机种子 `0, 1, 2, 3, 4` 重复生成合成数据，用于判断算法结论是否只在单个随机样本上成立。

## 文件与图片

### `repeat_recovery_error.png`

展示不同随机种子下各算法的恢复误差。

- 用途：比较算法恢复质量的稳定性。
- 主要结论：ISTA、FISTA 和 ADMM 在每个种子下的结果几乎重合，平均恢复误差均约为 `0.371`；Subgradient 平均恢复误差为 `0.799`，且波动更大。
- 报告/PPT 建议：用于支持“近端算法和 ADMM 的结果稳定且一致”这一结论。

### `repeat_nnz.png`

展示不同随机种子下各算法最终解的非零系数数量。

- 用途：比较算法产生稀疏解的稳定性。
- 主要结论：ISTA、FISTA 和 ADMM 的平均 nnz 约为 `19`，会随数据变化合理波动；Subgradient 在所有种子下 nnz 均为 `200`，没有产生有效稀疏解。

### `repeat_detail.csv`

保存每个随机种子、每种算法的逐次实验结果。

### `repeat_summary.csv`

保存每种算法各指标的均值和标准差，适合直接用于报告表格。

## 可直接使用的结论

> 五次独立重复实验表明，ISTA、FISTA 和 ADMM 的目标值、恢复误差与稀疏度高度一致，说明实验结论具有较好的稳定性。ADMM 平均仅需 61 次迭代，明显少于 ISTA 和 FISTA 的约 221 次。
