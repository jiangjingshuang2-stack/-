# ADMM Rho 敏感性实验说明

本实验固定 `lambda=0.1`，比较 `rho=0.1, 0.5, 1.0, 2.0, 5.0, 10.0` 时 ADMM 的表现。

## 文件与图片

### `rho_objective.png`

展示不同 rho 下 ADMM 的最终目标函数值。

- 用途：判断 rho 是否改变最终求解结果。
- 主要结论：所有 rho 的最终目标值均约为 `0.554998`，恢复误差均约为 `0.389504`，并且 nnz 都为 `17`。因此 rho 几乎不改变最终解，主要影响收敛过程。
- 报告/PPT 建议：用于说明 ADMM 参数 rho 影响效率，但不改变原问题的最优解。

### `rho_primal_dual_residual.png`

展示不同 rho 下原始残差和对偶残差随迭代变化的曲线。

- 用途：分析 ADMM 的收敛速度及原始、对偶可行性之间的平衡。
- 读图：两类残差越快同时下降到较小值，说明 rho 设置越合适。
- 主要结论：`rho=1.0` 和 `rho=2.0` 收敛最快，分别使用 `59` 和 `63` 次迭代。rho 太小或太大都会减慢收敛；`rho=0.1` 在 `500` 次迭代内仍未满足停止条件。
- 报告/PPT 建议：优先使用此图展示 rho 对 ADMM 收敛速度的影响。

### `rho_sensitivity.csv`

保存每个 rho 的目标值、恢复误差、nnz、迭代次数、运行时间和最终残差。

## 可直接使用的结论

> Rho 对 ADMM 最终解的影响很小，但对收敛速度影响明显。本实验中 rho=1 或 rho=2 的迭代效率最佳，过小或过大的 rho 都会导致收敛变慢。
