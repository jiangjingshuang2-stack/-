# 合成数据基础实验说明

本实验在已知真实稀疏系数的合成数据上比较 Subgradient、ISTA、FISTA 和 ADMM。主要参数为 `m=80`、`n=200`、真实非零系数数目 `10`、`lambda=0.1`、`max_iter=500`。

## 文件与图片

### `synthetic_objective.png`

展示四种算法的目标函数值随迭代次数变化的收敛曲线。

- 用途：比较算法是否收敛，以及达到稳定目标值所需的迭代次数。
- 读图：曲线越快下降并趋于稳定，说明算法迭代收敛越快。
- 主要结论：ISTA、FISTA 和 ADMM 最终目标值均约为 `0.5550`；ADMM 仅用 `59` 次迭代，ISTA 和 FISTA 分别使用 `176` 和 `168` 次。Subgradient 在 `500` 次迭代后目标值仍较高。
- 报告/PPT 建议：用于说明近端算法和 ADMM 比普通次梯度法更适合求解 LASSO。

### `synthetic_coefficients.png`

分别对比四种算法恢复出的系数与真实稀疏系数。

- 用途：直观观察算法能否识别真实非零特征，并抑制无关特征。
- 读图：估计曲线与黑色真实信号越接近，稀疏恢复效果越好。
- 主要结论：ISTA、FISTA 和 ADMM 给出几乎相同的稀疏解，均保留 `17` 个非零系数；Subgradient 保留了全部 `200` 个系数，未产生有效稀疏性。
- 注意：所有算法的 `support_ok=False`，因为该指标要求估计支撑集与真实支撑集完全一致，判定非常严格。
- 报告/PPT 建议：用于展示 L1 正则化产生稀疏解的效果，以及 Subgradient 与近端方法的差异。

### `synthetic_summary.csv`

包含最终目标值、恢复误差、非零系数数目、迭代次数、运行时间和 ADMM 残差。

## 可直接使用的结论

> 在相同参数下，ISTA、FISTA 和 ADMM 收敛到几乎相同的 LASSO 解，其恢复误差约为 0.390，明显优于次梯度法的 0.425。ADMM 所需迭代次数最少，而次梯度法未能产生稀疏解。
